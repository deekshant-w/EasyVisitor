About

Easy Counter is a simple counter app for django unlike several other complicated and difficult to setup apps which require a lot of time and concentration to set up

I created this app-middleware because I couldnt find an easy to setup package for this task hence I thought that it might be same for others too.

The features of this app are quite less as of the day I am writtng it but if anyone needs some other features too then let me know, I might add them for everyone who uses this app in future.
